Formato del archivo

El archivo debe ser .xlsx con el mismo formato que la tabla de cuentas corrientes:

Cliente Saldo Vencido Balance/Especial Ene-25 Feb-25 ...
Nombre del cliente Se ignora 1500.00 200.00 300.00 450.00 ...
  • La primera fila se interpreta como encabezado y se omite.
  • La columna Saldo se ignora ya que es un valor calculado.
  • Las columnas de meses deben tener formato Mes-Año (ej: Ene-25, Feb-25, Mar-25).
  • Si los periodos no existen en la configuración, se crearán automáticamente.
  • Si un cliente ya existe, se actualizarán sus valores.
  • Puede usar el archivo generado por "Exportar a Excel" como plantilla.
Atención: Esta funcionalidad está diseñada para la importación inicial de datos. Los clientes existentes serán actualizados con los valores del archivo. Asegúrese de que los datos sean correctos antes de proceder.
